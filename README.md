# CS50 Finance

This repository contains my personal solution to the Finance problem from Harvard University's CS50x course.

## Description

This project implements a stock trading web application where users can register, manage a virtual portfolio, buy and sell stocks, and track transaction history.

## Language

- Python
- HTML
- CSS
- SQL

## Technologies

- Flask
- SQLite
- Jinja
- Bootstrap

## Concepts Practiced

- Web Development
- Databases
- Authentication
- REST APIs
- SQL
- Flask

## Features

- User registration and authentication
- Buy and sell stocks
- Portfolio management
- Transaction history
- Real-time stock quotes
